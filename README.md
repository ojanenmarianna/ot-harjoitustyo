# Tetris

Tetris-pelissä tarkoituksena on järjestää erimuotoiset palikat niin, että niistä muodostuu kokonaisia vaakarivejä. Kun vaakarivi täyttyy, se poistetaan 
peliruudukosta,jolloin ruudukkoon vapautuu rivi. Peli jatkuu, kunnes ruudukkoon ei enää mahdu uutta palikkaa.
